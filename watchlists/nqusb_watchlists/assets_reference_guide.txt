USE CASES
=========

1. ASSET ALLOCATION SCREENING
   - Run screener on "Asset Classes" to compare stocks/bonds/commodities/crypto
   - Example ratios: SPY/TLT (stocks vs bonds), GLD/SPY (gold vs equities)

2. SECTOR ROTATION
   - Screen "SPY Sectors" to find strongest/weakest sectors
   - Compare XLY/XLP (risk-on vs risk-off)

3. YIELD CURVE ANALYSIS
   - Screen "Government Bonds" to analyze curve shape
   - Ratios: TLT/SHY (steepness), IEF/BIL (intermediate curve)

4. SIZE & STYLE FACTORS
   - Screen "Main Indexes" for factor performance
   - Compare IWO/IWN (small growth vs value), SPYG/SPYV (large growth vs value)

5. COMMODITY TRENDS
   - Screen "Commodities" for inflation indicators
   - Track GLD/SLV ratio, USO trends, DBA for agriculture
