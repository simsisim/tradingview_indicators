USE CASES
=========

1. FACTOR ROTATION STRATEGY
   Monitor value/growth ratios across cap sizes:
   - IWD/IWF, SPYV/SPYG (large cap)
   - MDYV/MDYG (mid cap)
   - IJS/IJT, IWN/IWO (small cap)
   Screen all 5 together to identify which factor is leading

2. RISK MANAGEMENT
   Track risk-on/off indicators:
   - XLP/XLY (consumer defensive vs cyclical)
   - SPY/TLT (stocks vs bonds)
   - DJU/DJT (utilities vs transportation)
   Declining values = Increasing risk appetite

3. SECTOR ROTATION
   Screen all 11 sector/SPY ratios:
   - Rising = Sector outperforming market
   - Falling = Sector underperforming market
   Sort by return % to find strongest/weakest sectors

4. YIELD CURVE ANALYSIS
   Monitor curve shape:
   - TLT/SHY (overall steepness)
   - IEF/BIL (short-intermediate)
   - TLH/IEI (intermediate-long)
   Flattening = Economic slowdown concerns

5. COMMODITY TRENDS
   Track commodity leadership:
   - DBA/DJP, USO/DJP, GLD/DJP, SLV/DJP
   - GLD/SLV ratio for precious metals
   Screen to identify hot commodity sectors

6. MULTI-ASSET ALLOCATION
   Screen inter-asset ratios for allocation signals:
   - SPY/TLT, SPY/GLD for stock positioning
   - BTC/GLD for crypto vs traditional safe haven
   - DXY/VBINX for dollar strength impact

SCREENER CONFIGURATION TIPS
============================

1. For Pine Screener, create separate watchlists by category:
   - "Value-Growth Ratios" (5 symbols)
   - "Risk Indicators" (8 symbols: risk-on/off + inter-asset)
   - "Sector Leadership" (11 symbols)
   - "Bond Market" (12 symbols: govt + corp + curve)
   - "Commodities" (11 symbols: commodity + spreads)

2. Best timeframes for analysis:
   - Daily: Tactical signals (1W, 1M returns)
   - Weekly: Intermediate trends (1M, 3M, 6M returns)
   - Monthly: Strategic positioning (6M, 1Y, YTD returns)

3. Key RS benchmarks to use:
   - Most ratios: Use SPY as benchmark for RS rating
   - Bond ratios: Use TLT or AGG
   - Commodity ratios: Use DJP

4. Sorting strategies:
   - Sort by 1M Return % descending: Find recent momentum
   - Sort by RS 1Y descending: Find long-term strength
   - Sort by YTD Return %: Identify year-to-date leaders
